# Flutter Information Management App

A modern Flutter application for efficient information management.

## Features
- Modern UI
- CRUD Operations
- Responsive Design
- Flutter & Dart

## Installation
```bash
git clone https://github.com/tanishqkolhatkar93/Flutter_Info_Managment_App.git
cd Flutter_Info_Managment_App
flutter pub get
flutter run
```
